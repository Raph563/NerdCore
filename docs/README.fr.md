# NerdCore - Documentation (FR)

NerdCore est la couche partagee obligatoire pour `StatNerd` et `ProductHelper`.

## Fonctionnalites principales
- bootstrap runtime commun des addons
- aide de detection de langue
- stockage centralise des reglages communs
- integration au menu Parametres Grocy (en haut a droite)
- page dediee: `/stocksettings?nerdcore=1`

## Ordre d'installation
1. Installer NerdCore
2. Installer StatNerd et/ou ProductHelper

## Fichiers geres
- `config/data/custom_js_nerdcore.html`
- `config/data/custom_js.html`
- `config/data/nerdcore-addon-state.json`
