# NerdCore - Documentation (EN)

NerdCore is the mandatory shared layer for `StatNerd` and `ProductHelper`.

## Main features
- shared addon runtime bootstrap
- language detection helper API
- common settings storage
- Grocy settings menu integration (top-right)
- dedicated settings page: `/stocksettings?nerdcore=1`

## Install order
1. Install NerdCore
2. Install StatNerd and/or ProductHelper

## Managed files
- `config/data/custom_js_nerdcore.html`
- `config/data/custom_js.html`
- `config/data/nerdcore-addon-state.json`
